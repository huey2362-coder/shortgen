# ShortGen — 사용자가 할 것 (그 "1%")

> 나머지(코드·빌드·운영)는 전부 클로드. 아래는 **신원·계정·돈**이 걸려 클로드가 못 하는 지점만.

## 0. 지금 당장 로컬에서 돌려보기 (계정 0)
```
cd D:\WORK\shortgen_saas
python -m uvicorn app.main:app --port 8099
```
브라우저에서 http://127.0.0.1:8099 → 대본 넣고 Generate → 영상 나옴. **무료(그래디언트) 티어는 지금 완전 동작.**

## 1. 실사 B롤(Pro 배경) 켜기 — Pexels 무료 키 (1분)
- pexels.com/api → 가입 → API 키 복사
- 웹 UI에서 Background = "Real footage" 선택 후 키 붙여넣기 (또는 배포 시 env)
- 없으면 무료 그래디언트만 쓰면 됨. **급하지 않음.**

## 2. 유료 구독(수익) 켜기 — Stripe (결제가 자동으로 들어오는 지점)
- stripe.com 가입(신원·은행 연결) → 대시보드에서:
  - **Pro 구독 상품** 생성 → `price_...` 복사
  - **Secret key** `sk_...` 복사
  - (배포 후) **Webhook** 추가 → `whsec_...`
- 이 3개를 배포 환경변수에 넣으면 `app/billing.py` 가 활성 → 체크아웃/구독 동작
- **돈은 여기(Stripe→네 은행)로 들어온다. 이 계정·은행연결만 너.**

## 3. 인터넷에 올리기 — 배포 (호스팅 계정)
- Docker 이미지 준비됨(`Dockerfile`). 아무 컨테이너 호스팅(Railway·Render·Fly 등, 무료/저가 티어) 계정 하나
- 도메인 1개(선택)
- 클로드가 배포 스크립트·설정 다 써줌. **계정 로그인·결제만 너.**

## 4. 이후 = 거의 0
- 입금 받기(Stripe→은행) + 가끔 승인. 운영·개선·마케팅 콘텐츠는 클로드.

---
### 요약: 네가 만들 계정 = Pexels(무료·선택) / Stripe(수익) / 호스팅 1곳. 그게 전부다.
### 클로드가 못 넘는 벽 = 이 계정 생성과 은행연결(신원). 나머지 100% 내가.
